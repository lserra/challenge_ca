Qty Accidents in Brazil - Top Cities¶

This exercise requires all necessary code to be written in any programming language (Preferably Python or Java). The goal is to retrieve the metric of municipal work related accidents provided by the data based on the Public Brazilian Social Security API ( http://api.dataprev.gov.br/doc/index.html ). As a first step, retrieve data for all years saving it in a .csv file and then transform and analyze the data to get the top 3 brazilian cities on the dataset, which grew the metric the most between 2003 and 2007, by geopolitical region.
